About

The Document Tracker is an application designed to help users manage and track their documents efficiently. It allows users to create, edit, and delete documents, and keeps a record of the document's edit history.

The application is built using Angular for the frontend and Node.js with MongoDB for the backend. It provides a user-friendly interface for managing documents and is suitable for personal and professional use.
